## Стек технологий

## Инструкция по запуску

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/Nelson1o/cat-pinterest.git
   ```
2. Установить зависимости:
   ```bash
   npm install
   ```
3. Запустить режим разработки:
   ```bash
   npm start
   ```
