# Cat Pinterest

- Live-демо: https://nelson1o.github.io/cat-pinterest/gallery

## Стек технологий

- React
- TypeScript
- Vite
- Tanstack Query
- SCSS Modules

## Запуск приложения

1. Клонируете репозиторий:
   ```bash
   git clone https://github.com/Nelson1o/cat-pinterest.git
   ```
2. Устанавливаете зависимости:
   ```bash
   npm install
   ```
3. Запустите приложение:
   ```bash
   npm start
   ```
